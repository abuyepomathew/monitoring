# Electrical Load Monitoring System

## 📌 Project Description
This is a C++ console-based Electrical Load Monitoring System.
It allows users to register appliances, calculate energy consumption,
and generate monthly billing summaries.

## 🚀 Features
- Register appliances
- View all appliances
- Search appliance
- Remove appliance
- Monthly billing summary
- File saving using appliances.txt
- Billing report generation

## 🛠️ Technologies Used
- C++
- Object-Oriented Programming (OOP)
- File Handling

## ▶️ How to Compile and Run

Compile:
g++ main.cpp -o monitoring.exe

Run:
.\monitoring.exe
